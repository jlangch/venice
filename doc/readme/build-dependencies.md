# Build Dependencies


## Gradle

```groovy
dependencies {
    compile 'com.github.jlangch:venice:1.10.18'
}
```

## Maven

```xml
<dependencies>
    <dependency>
        <groupId>com.github.jlangch</groupId>
        <artifactId>venice</artifactId>
        <version>1.10.18</version>
    </dependency>
</dependencies>
```
