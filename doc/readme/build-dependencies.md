# Build Dependencies


## Gradle

```groovy
dependencies {
    compile 'com.github.jlangch:venice:1.9.6'
}
```

## Maven

```xml
<dependencies>
    <dependency>
        <groupId>com.github.jlangch</groupId>
        <artifactId>venice</artifactId>
        <version>1.9.6</version>
    </dependency>
</dependencies>
```
